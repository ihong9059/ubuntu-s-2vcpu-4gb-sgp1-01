시스템 개요

SAM AI v3는 Flask 백엔드와 Vanilla JavaScript 프론트엔드로 구성된 멀티 AI 채팅 애플리케이션입니다.

전체 아키텍처
Browser
(Vanilla JS)
↔
Flask Server
(Python)
↔
AI Providers
(Claude, GPT, Gemini, Ollama)
JSON Storage
(대화 기록)

API 엔드포인트

채팅 API

Method Endpoint 설명
POST /api/chat Single 모드 채팅 (SSE 스트리밍)
POST /api/chat/multi Compare 모드 (다중 AI 병렬)
POST /api/chat/smart Smart 모드 (수집 + 통합)
POST /api/chat/stop 응답 생성 중단

대화 관리 API

Method Endpoint 설명
GET /api/conversations 대화 목록 조회
POST /api/conversations 새 대화 생성
GET /api/conversations/<id> 특정 대화 조회
PUT /api/conversations/<id> 대화 수정
DELETE /api/conversations/<id> 대화 삭제

설정 API

Method Endpoint 설명
GET /api/config 설정 조회 (API 키 마스킹)
POST /api/config/keys API 키 저장
GET /api/providers/status 프로바이더 상태 조회

실시간 스트리밍 (SSE)

Server-Sent Events를 사용하여 AI 응답을 실시간으로 전송합니다.

SSE 이벤트 형식
event: token
data: {"content": "Hello", "provider": "claude"}

event: done
data: {"provider": "claude", "full_response": "Hello, world!"}

event: error
data: {"error": "API key invalid", "provider": "claude"}

SSE 특징

  • 단방향: 서버에서 클라이언트로만 전송
  • 자동 재연결: 연결 끊김 시 자동 복구
  • 가벼움: WebSocket보다 단순하고 가벼움

기술 스택

백엔드

기술 용도
Python 3.9+ 서버 런타임
Flask 웹 프레임워크
Flask-CORS 크로스 도메인 지원
anthropic, openai, google-generativeai AI SDK

프론트엔드

기술 용도
Vanilla JavaScript UI 로직 (프레임워크 없음)
CSS3 다크 모드 스타일링
Marked.js 마크다운 렌더링
Highlight.js 코드 하이라이팅
Service Worker PWA, 오프라인 캐싱
