# 학생 진학 컨설팅 서비스 사업계획서

## 1. 사업 개요

### 1.1 서비스 명칭 (예시)
**"에듀패스(EduPath)" - 맞춤형 진학 컨설팅 서비스**

### 1.2 사업 목적
- 학생 개인의 특성과 현재 수준을 정확히 진단
- 맞춤형 학습 방향 및 진학 목표 설정
- 효율적인 학습 전략 수립을 통한 진학 성공률 향상

### 1.3 핵심 가치
- **개인화**: 모든 학생은 다르다는 전제하에 1:1 맞춤 설계
- **과학적 접근**: 데이터 기반 진단 및 분석
- **지속적 관리**: 일회성이 아닌 장기적 코칭

---

## 2. 서비스 프로세스

### 2.1 1단계: 메타인지 진단
| 항목 | 내용 |
|------|------|
| 목표 | 학생이 자신을 얼마나 정확히 파악하는지 확인 |
| 방법 | 자기평가 설문, 학습일지 분석, 면담 |
| 산출물 | 메타인지 리포트 |

### 2.2 2단계: 학업 수준 파악
| 항목 | 내용 |
|------|------|
| 목표 | 현재 학업 성취도 및 잠재력 측정 |
| 방법 | 과목별 진단평가, 내신/모의고사 분석, 학습습관 체크 |
| 산출물 | 학업역량 분석표 |

### 2.3 3단계: 성향 분석 (MBTI + α)
| 항목 | 내용 |
|------|------|
| 목표 | 학습 스타일 및 적합 진로 탐색 |
| 방법 | MBTI, 홀랜드 직업흥미검사, 학습유형검사 |
| 산출물 | 성향분석 리포트 |

### 2.4 4단계: 맞춤 솔루션 제공
- **학습방향 설정**: 강점 강화 vs 약점 보완 전략
- **목표 수립**: 단기(3개월)/중기(1년)/장기(입시) 목표
- **실행 로드맵**: 월별/주별 학습 계획표

| 운영/마케팅 | 사업 운영, 홍보 | 1명 |

- [ ] 상담 기록 및 리포트 생성 시스템
- [ ] 온/오프라인 상담 환경 구축

## 4. 추가 검토 사항 및 아이디어 제안

### 4.1 추가 검토 필요 사항

**1) 타겟 고객 명확화**
- 중학생 vs 고등학생 vs 재수생
- 수시 vs 정시 집중 여부
- 지역 범위 (온라인 전국 vs 오프라인 특정 지역)

**2) 차별화 전략**
- 기존 입시학원/컨설팅과의 차별점 명확화 필요
- 가격 정책 수립 (고가 프리미엄 vs 대중화)

**3) 성과 측정 기준**
- 서비스 효과를 어떻게 입증할 것인가?
- 진학 성공률, 성적 향상률 등 KPI 설정

### 4.2 추가 아이디어 제안

**1) AI 기반 학습 분석 도입**
- 학습 데이터를 AI로 분석하여 더 정밀한 진단 제공
- 경쟁력 있는 차별화 포인트

**2) 학부모 동반 상담 프로그램**
- 학생-학부모 간 진학 목표 합의 유도
- 가정에서의 학습 환경 개선 코칭

**3) 멘토링 연결 서비스**
- 목표 대학 재학생/졸업생과 연결
- 실제 경험담을 통한 동기부여

**4) 단계별 패키지 상품화**
```
[진단 패키지] 3가지 검사 + 종합 리포트 + 1회 상담
[코칭 패키지] 진단 + 3개월 학습관리 + 월 2회 상담
[프리미엄 패키지] 진단 + 6개월~1년 전담 관리
```

**5) 정기 세미나/설명회**
- 입시 트렌드, 학습법 등 무료 세미나로 잠재 고객 확보
- 온라인 웨비나로 확장 가능

**6) 포트폴리오/자기소개서 컨설팅**
- 수시 전형 대비 서류 작성 지원
- 추가 수익원 + 서비스 완결성

## 6. 결론

본 서비스는 메타인지 진단, 학업 수준 파악, 성향 분석의 3단계 진단을 통해 학생 개인에게 최적화된 학습 방향과 목표를 설정해주는 맞춤형 진학 컨설팅 서비스입니다. 과학적이고 체계적인 접근을 통해 학생들의 진학 성공을 지원하고, 지속적인 관리를 통해 장기적인 성과를 창출하는 것을 목표로 합니다.
