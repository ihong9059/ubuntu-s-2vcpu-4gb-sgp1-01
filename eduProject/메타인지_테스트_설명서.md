# 고등학교 2학년 대상 메타인지 테스트 설명서

## 개요

### 메타인지란?
메타인지(Metacognition)는 **"자신의 인지 과정에 대해 아는 것"**, 즉 자신이 무엇을 알고 무엇을 모르는지 정확히 파악하는 능력입니다. 학습에서 메타인지가 높은 학생은 효율적인 학습 전략을 선택하고, 자신의 약점을 정확히 인식하여 보완할 수 있습니다.

### 테스트 목적
- 학생이 자신의 학업 수준을 얼마나 정확히 인식하는지 파악
- 학습 계획 및 전략 수립 능력 진단
- 자기 점검 및 조절 능력 평가
- 맞춤형 학습 코칭의 기초 자료 확보

### 대상
- 고등학교 2학년 (대학 진학 준비생)

### 소요 시간
- 전체 테스트: 약 60~90분
- 결과 분석 면담: 약 30분

---

## 테스트 구성

메타인지 테스트는 **5개 영역**으로 구성됩니다.

| 영역 | 측정 내용 | 방법 |
|------|----------|------|
| 1. 자기인식 정확도 | 예측 vs 실제 성적 비교 | 예측 테스트 |
| 2. 학습전략 인식 | 학습 방법에 대한 이해도 | 설문지 |
| 3. 계획 및 목표설정 | 학습 계획 수립 능력 | 과제 수행 |
| 4. 자기점검 능력 | 학습 중 모니터링 능력 | 씽크어라우드 |
| 5. 자기조절 능력 | 전략 수정 및 조절 능력 | 면담 + 설문 |

### 영역 1: 자기인식 정확도 테스트

#### 목적
학생이 자신의 실력을 얼마나 정확하게 파악하고 있는지 측정합니다.

#### 방법: 예측-수행-비교 테스트

**[절차]**

1. **예측 단계** (10분)
   - 학생에게 과목별(국어, 수학, 영어, 탐구) 10문제씩 제시
   - 문제를 풀기 전, 각 문제에 대해 맞출 자신감을 표시

   ```
   예시 양식:
   ┌─────────────────────────────────────────────────┐
   │ 문제 1번: 맞출 자신감은?                          │
   │ □ 확실히 맞음 (90%↑)                             │
   │ □ 아마 맞을 것 (70~89%)                          │
   │ □ 반반 (50~69%)                                  │
   │ □ 틀릴 가능성 높음 (30~49%)                       │
   │ □ 모르겠음 (30%↓)                                │
   └─────────────────────────────────────────────────┘
   ```

2. **수행 단계** (30분)
   - 학생이 실제로 문제를 풂

3. **비교 단계** (분석)
   - 예측과 실제 결과를 비교하여 **자기인식 정확도 점수** 산출

**[분석 기준]**

| 유형 | 설명 | 특징 |
|------|------|------|
| 정확형 | 예측 ≈ 실제 | 메타인지 양호, 현실적 목표 설정 가능 |
| 과대평가형 | 예측 > 실제 | 자신감 과잉, 학습량 부족 가능성 |
| 과소평가형 | 예측 < 실제 | 자신감 부족, 불안 성향 가능성 |

**[기록 양식]**

```
학생명: ________________  검사일: ________________

과목: 국어
┌────┬──────────┬──────────┬──────────┐
│문항│ 예측자신감 │ 실제정답 │   차이   │
├────┼──────────┼──────────┼──────────┤
│ 1  │   90%    │    O     │  일치    │
│ 2  │   70%    │    X     │  과대    │
│ 3  │   50%    │    O     │  과소    │
│... │   ...    │   ...    │   ...    │
└────┴──────────┴──────────┴──────────┘

자기인식 정확도: _____ %
유형 판정: □ 정확형  □ 과대평가형  □ 과소평가형
```

### 영역 4: 자기점검 능력 (씽크어라우드)

#### 목적
학습 과정에서 스스로 이해도를 점검하는 능력을 평가합니다.

#### 방법: Think-Aloud (생각 말하기) 기법

**[절차]**

1. **문제 제시**
   - 학생의 취약 과목에서 중상 난이도 문제 1개 선정
   - 예: 수학 문제, 국어 지문 분석, 영어 독해 등

2. **지시문 안내**
   ```
   "지금부터 이 문제를 풀면서 머릿속에 떠오르는 생각을
    모두 말로 표현해 주세요.

   예를 들어:
   - '이 부분이 이해가 안 돼'
   - '아, 이건 공식을 적용하면 되겠다'
   - '여기서 막히네, 다른 방법을 찾아봐야겠다'

   틀려도 괜찮으니 생각하는 과정을 자연스럽게 말해주세요."
   ```

3. **관찰 및 기록** (컨설턴트)
   - 학생의 발화 내용을 기록
   - 아래 체크리스트로 평가

**[관찰 체크리스트]**

```
학생명: ________________  검사일: ________________

■ 자기점검 행동 관찰 기록

[이해 점검]
□ 문제를 읽고 무엇을 묻는지 파악하려 함
□ 이해 안 되는 부분을 인식하고 표현함
□ 핵심 정보를 구분하려는 시도를 함

[전략 사용]
□ 문제 해결에 필요한 개념/공식을 떠올림
□ 풀이 방법을 여러 가지 고려함
□ 막히면 다른 접근법을 시도함

[오류 감지]
□ 자신의 답에 대해 확신/불확신을 표현함
□ 풀이 과정의 오류를 스스로 발견함
□ 답을 검토하거나 재확인함

[자기 조절]
□ 어려움을 느꼈을 때 감정을 조절함
□ 포기하지 않고 끝까지 시도함
□ 모르는 것을 인정하고 표현함

■ 관찰 점수 (각 3점 만점, 총 36점)

이해 점검: ___/9점

오류 감지: ___/9점
자기 조절: ___/9점
─────────────────
총    점: ___/36점

■ 관찰 특이사항:
_________________________________________________
_________________________________________________
```

### 영역 5: 자기조절 능력

#### 목적
학습 과정에서 전략을 수정하고 조절하는 능력을 평가합니다.

#### 방법: 시나리오 기반 면담 + 설문

**[면담 질문]** (10분)

```
■ 과거 학습 경험 탐색

1. "지난 시험에서 가장 잘 본 과목은 무엇이고,
    왜 잘 봤다고 생각하나요?"
   → (원인 분석 능력 확인)

2. "반대로 가장 못 본 과목은요?
    원인이 무엇이었다고 생각하나요?"
   → (실패 원인 분석 능력 확인)

3. "그 과목 성적을 올리기 위해
    어떤 노력을 했거나 할 계획인가요?"
   → (전략 수정 능력 확인)

4. "공부하다가 집중이 안 될 때
    어떻게 하나요?"
   → (자기조절 방법 확인)

5. "계획대로 공부가 안 됐던 적이 있나요?
    그때 어떻게 대처했나요?"
   → (계획 수정 능력 확인)
```

## 후속 조치

### 결과 피드백 면담 (30분)

1. **결과 공유** (10분)
   - 각 영역별 점수와 의미 설명
   - 강점과 개선점 안내

2. **원인 탐색** (10분)
   - 학생과 함께 낮은 점수 영역의 원인 논의
   - 학생의 의견 청취

3. **개선 방향 합의** (10분)
   - 구체적 개선 목표 설정
   - 다음 상담 일정 수립

   - 시험이 아님을 강조
   - 솔직한 응답이 중요함을 안내

2. **충분한 시간 제공**
   - 서두르지 않도록 안내

   - 힌트나 유도 금지
   - 격려는 하되 답을 알려주지 않음
