# 고등학교 2학년 메타인지 분석 분류 체계

## 1. 2대 영역 모델 (Schraw & Dennison)

| 대영역 | 하위영역 |
|--------|----------|
| **메타인지적 지식** | 선언적 지식, 절차적 지식, 조건적 지식 |
| **메타인지적 조절** | 계획, 점검(모니터링), 평가, 조절, 정보관리 |

### 메타인지적 지식 세부 설명
- **선언적 지식**: 자신의 학습자로서의 특성, 전략에 대한 지식
- **절차적 지식**: 전략을 어떻게 사용하는지에 대한 지식
- **조건적 지식**: 언제, 왜 특정 전략을 사용하는지에 대한 지식

### 메타인지적 조절 세부 설명
- **계획**: 학습 목표 설정, 시간 배분
- **점검(모니터링)**: 학습 과정 중 이해도 확인
- **평가**: 학습 결과 점검
- **조절**: 전략 수정 및 조정
- **정보관리**: 학습 자료 조직 및 관리

## 2. 학습 맥락에서 자주 사용되는 5영역

| 영역 | 설명 |
|------|------|
| 계획(Planning) | 학습 목표 설정, 시간 배분 |
| 점검(Monitoring) | 학습 과정 중 이해도 확인 |
| 조절(Regulation) | 전략 수정 및 조정 |
| 평가(Evaluation) | 학습 결과 점검 |
| 자기인식(Self-awareness) | 자신의 강약점 파악 |
