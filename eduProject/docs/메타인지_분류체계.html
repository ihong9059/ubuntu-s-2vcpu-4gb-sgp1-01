메타인지적 지식 세부 설명

선언적 지식 (Declarative Knowledge)

자신의 학습자로서의 특성, 전략에 대한 지식

예: "나는 아침에 집중이 잘 된다", "요약 정리가 암기에 효과적이다"
절차적 지식 (Procedural Knowledge)

전략을 어떻게 사용하는지에 대한 지식

예: "마인드맵을 그리는 방법", "오답노트 작성 순서"
조건적 지식 (Conditional Knowledge)

언제, 왜 특정 전략을 사용하는지에 대한 지식

예: "수학은 문제풀이, 국어는 지문 분석이 중요하다"

메타인지적 조절 세부 설명

계획 (Planning)

학습 목표 설정, 시간 배분

점검 (Monitoring)

학습 과정 중 이해도 확인

평가 (Evaluation)

학습 결과 점검

조절 (Regulation)

전략 수정 및 조정

정보관리 (Information Management)

학습 자료 조직 및 관리

학습 맥락에서 자주 사용되는 5영역

영역 설명
계획 (Planning) 학습 목표 설정, 시간 배분
점검 (Monitoring) 학습 과정 중 이해도 확인
조절 (Regulation) 전략 수정 및 조정
평가 (Evaluation) 학습 결과 점검
자기인식 (Self-awareness) 자신의 강약점 파악
5영역 모델의 활용

이 5영역 모델은 고등학교 학습 상황에 맞게 조정된 실용적인 분류입니다. 각 영역은 독립적이면서도 상호 연관되어 있어, 한 영역의 향상이 다른 영역에도 긍정적 영향을 줍니다.
