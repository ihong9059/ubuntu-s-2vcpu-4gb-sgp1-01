개요

메타인지란?

메타인지(Metacognition)는 "자신의 인지 과정에 대해 아는 것", 즉 자신이 무엇을 알고 무엇을 모르는지 정확히 파악하는 능력입니다. 학습에서 메타인지가 높은 학생은 효율적인 학습 전략을 선택하고, 자신의 약점을 정확히 인식하여 보완할 수 있습니다.

테스트 목적
  • 학생이 자신의 학업 수준을 얼마나 정확히 인식하는지 파악
  • 학습 계획 및 전략 수립 능력 진단
  • 자기 점검 및 조절 능력 평가
  • 맞춤형 학습 코칭의 기초 자료 확보
대상 및 소요시간
  • 대상: 고등학교 2학년 (대학 진학 준비생)
  • 전체 테스트: 약 60~90분
  • 결과 분석 면담: 약 30분

영역 1 자기인식 정확도 테스트

목적: 학생이 자신의 실력을 얼마나 정확하게 파악하고 있는지 측정

방법: 예측-수행-비교 테스트
1예측 단계
문제를 풀기 전, 각 문제에 대해 맞출 자신감을 표시 (10분)
2수행 단계
학생이 실제로 문제를 풀이 (30분)
3비교 단계
예측과 실제 결과를 비교하여 점수 산출
예시 양식: ┌─────────────────────────────────────────────────┐ │ 문제 1번: 맞출 자신감은? │ │ □ 확실히 맞음 (90%↑) │ │ □ 아마 맞을 것 (70~89%) │ │ □ 반반 (50~69%) │ │ □ 틀릴 가능성 높음 (30~49%) │ │ □ 모르겠음 (30%↓) │ └─────────────────────────────────────────────────┘
분석 기준
유형설명특징
정확형예측 ≈ 실제메타인지 양호, 현실적 목표 설정 가능
과대평가형예측 > 실제자신감 과잉, 학습량 부족 가능성
과소평가형예측 < 실제자신감 부족, 불안 성향 가능성

영역 2 학습전략 인식 설문

목적: 학생이 효과적인 학습 전략을 알고 있는지, 실제로 사용하는지 파악

방법: 자기보고식 설문지 (5점 척도, 총 20문항)

A. 계획 전략 (5문항)
1. 공부 전 계획 세우기
2. 시험 전 일정 정하기
3. 과목별 시간 배분
4. 목표 성적 설정
5. 하루 공부량 실천
B. 점검 전략 (5문항)
6. 이해도 스스로 확인
7. 오답 원인 분석
8. 어려운 부분 표시
9. 시험 후 오답 정리
10. 복습 시 내용 떠올리기
C. 조절 전략 (5문항)
11. 방법 변경 시도
12. 다른 자료 찾기
13. 휴식 후 재시작
14. 우선순위 정하기
15. 원인 파악 및 개선
D. 인지 전략 (5문항)
16. 밑줄/표시하기
17. 요약/정리노트
18. 기존 지식과 연결
19. 나만의 말로 설명
20. 그림/도표 활용

영역 3 계획 및 목표설정 능력

목적: 실제 학습 계획을 세우는 능력을 평가

방법: 계획 수립 과제 (15분)

과제 지시문

[상황] 3주 후 중간고사가 있습니다.

시험 과목: 국어, 수학, 영어, 한국사, 생명과학

[과제]
1. 3주간의 학습 계획표를 작성하세요.
2. 과목별 목표 성적을 적으세요.
3. 목표 달성을 위한 구체적 실행 방법을 적으세요.

평가 기준표 (컨설턴트용)
평가 항목배점평가 기준
목표의 구체성20점구체적 점수/등급 제시 여부
목표의 현실성20점이전 성적 대비 합리적 목표 설정
계획의 구체성20점시간, 분량, 방법 명시 여부
균형성20점과목별 적절한 시간 배분
실행 가능성20점실제 생활 패턴 고려 여부
총점100점

영역 4 자기점검 능력 (씽크어라우드)

목적: 학습 과정에서 스스로 이해도를 점검하는 능력을 평가

방법: Think-Aloud (생각 말하기) 기법

지시문 안내

"지금부터 이 문제를 풀면서 머릿속에 떠오르는 생각을 모두 말로 표현해 주세요.

예를 들어:
- '이 부분이 이해가 안 돼'
- '아, 이건 공식을 적용하면 되겠다'
- '여기서 막히네, 다른 방법을 찾아봐야겠다'

틀려도 괜찮으니 생각하는 과정을 자연스럽게 말해주세요."

관찰 체크리스트
이해 점검 (9점)
  • 문제를 읽고 무엇을 묻는지 파악하려 함
  • 이해 안 되는 부분을 인식하고 표현함
  • 핵심 정보를 구분하려는 시도를 함
전략 사용 (9점)
  • 필요한 개념/공식을 떠올림
  • 풀이 방법을 여러 가지 고려함
  • 막히면 다른 접근법을 시도함
오류 감지 (9점)
  • 확신/불확신을 표현함
  • 풀이 과정의 오류를 스스로 발견함
  • 답을 검토하거나 재확인함
자기 조절 (9점)
  • 어려움 시 감정을 조절함
  • 포기하지 않고 끝까지 시도함
  • 모르는 것을 인정하고 표현함

총점: 36점 만점

영역 5 자기조절 능력

목적: 학습 과정에서 전략을 수정하고 조절하는 능력을 평가

방법: 시나리오 기반 면담 + 설문

시나리오 설문 예시
상황 1

"수학 문제집을 풀었는데 10문제 중 7문제를 틀렸습니다."

A. 그냥 넘어가고 다음 단원으로 간다 1점
B. 틀린 문제만 답 보고 넘어간다 2점
C. 틀린 문제를 다시 풀고 분석한다 4점
D. 해당 개념부터 다시 공부한다 3점
점수 산정
선택점수의미
A1점회피/포기 성향
B2점소극적 대처
C4점적극적/전략적 대처
