## 시나리오 조합 규칙

### 최종 시나리오 = 공통(50) + 도시(20) + 목적(20) + 예산(15) = 105개

### 조합 우선순위
1. 공통 템플릿 (Day 1-2, Day 마지막)
2. 도시별 템플릿 (Day 3-8 관광)
3. 목적별 템플릿 (전체에 분산)
4. 예산별 템플릿 (숙소/식사/교통에 적용)

### 중복 방지 규칙
- 같은 장소에서 예산별/목적별 중복 시 → 예산별 우선
- 호텔 체크인: 예산별 버전만 사용 (공통에서 제외)
- 식당 주문: 예산별 버전만 사용 (공통에서 제외)
