사용 방법 안내

🎯 게임 목표

영어 단어를 학습하면서 전 세계 도시를 여행하는 게임입니다. 문제를 맞추면 다음 도시로 이동하고, 틀리면 뒤로 돌아갑니다. 모든 도시를 방문하여 세계일주를 완료하세요!

📚 난이도 선택

🌱 초보자 800 단어 / 4회 세계일주
🌿 중급자 1,200 단어 / 6회 세계일주
🌳 고급자 2,000 단어 / 10회 세계일주
🏆 전문가 2,000 단어 / 10회 세계일주

🗺️ 경로 선택

7개의 세계일주 경로 중 하나를 선택합니다. 각 경로는 200개 도시로 구성되어 있으며, 경도를 기준으로 지구를 순환하는 경로입니다.

  • 경로 A ~ G: 각각 다른 도시들로 구성
  • 경로 선택 시 지도에서 미리보기 가능
  • 10회 세계일주 시 완료된 경로도 다시 선택 가능

📍 출발 도시 선택

선택한 경로 내에서 원하는 출발 도시를 지도에서 클릭하여 선택합니다. 해당 도시부터 시작하여 경로를 따라 이동합니다.

🎮 게임 진행

  • 영어 → 한글: 영어 단어를 보고 한글 뜻을 선택
  • 한글 → 영어: 한글 뜻을 보고 영어 단어를 선택
  • 정답: 다음 도시로 이동! 이동 거리가 누적됩니다
  • 오답: 1~2개 도시 뒤로 이동

🔊 발음 듣기

  • 영어 → 한글 문제: 바로 발음 듣기 가능
  • 한글 → 영어 문제: 정답 후 발음 듣기 가능
  • 다음 도시 이동 시 도시 이름 음성 안내

💾 저장 및 이어하기

  • 게임 중 "일시정지 및 저장" 버튼으로 저장
  • 정답/오답 시 자동 저장
  • 메인 화면에서 "이어하기"로 계속 진행

📊 화면 정보

  • 방문 도시: 현재 경로에서 방문한 도시 수
  • 정답률: 현재 세계일주의 정답률
  • 이번 경로 거리: 현재 세계일주에서 이동한 거리
  • 총 누적 거리: 모든 세계일주 총 이동 거리

💡 팁

  • 발음 듣기를 활용하여 영어 발음도 함께 학습하세요
  • 틀린 문제는 다시 나오니 반복 학습이 됩니다
  • 지도를 보며 세계 지리도 함께 익힐 수 있습니다
