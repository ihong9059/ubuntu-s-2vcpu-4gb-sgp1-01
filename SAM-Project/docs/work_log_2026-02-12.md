### 구현 내용
1. **프로젝트 구조** - 전체 디렉토리 및 파일 구조 생성
2. **백엔드 (Flask)**
   - `app.py` - 메인 Flask 앱 (포트 5060)
   - `config.py` - 설정 상수
   - `services/gemini_service.py` - Gemini API SSE 스트리밍

   - `services/conversation_manager.py` - 대화 저장/관리
   - `services/config_manager.py` - API 키/설정 관리
   - `services/news_service.py` - RSS 뉴스 + 생일/일정
   - `services/member_data.py` - 8명 멤버 프로필 + 시스템 프롬프트
3. **프론트엔드**
   - K-pop 다크 퍼플 테마 CSS
   - 멤버 선택 바 (8명, 색상 자동 전환)
   - 채팅 UI (SSE 스트리밍, 마크다운 렌더링)
   - 대화 목록 (사이드바)
   - 음성 입력 (Web Speech API) + 음성 출력 (edge-tts)
   - 뉴스 패널 (생일 D-day, 일정, RSS 뉴스)
   - 설정 모달 (API 키 입력)
   - PWA (manifest.json + Service Worker)
   - 반응형 레이아웃 (iPad/모바일)
4. **의존성** - flask, flask-cors, google-generativeai, edge-tts, feedparser
