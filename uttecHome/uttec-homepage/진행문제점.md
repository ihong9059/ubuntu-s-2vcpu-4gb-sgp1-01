## 2. 현재 문제점 및 개선 필요사항

### 2.1 콘텐츠 관련 문제

#### 2.1.1 기술력/인력 정보 미반영
- **문제**: 홍광선 대표(40년 경력), 임호균 이사(38년 경력)의 상세 기술력이 About 페이지에 충분히 반영되지 않음
- **해결방안**: /about 페이지에 핵심 인력 소개 섹션 추가
- **참고자료**: 이력서_홍광선.pdf, 경력서 - 임호균.pdf

#### 2.1.2 데모 사이트 URL 불일치
- **문제**: demos.ts의 demoUrl이 상대경로(/demo/xxx/)로 되어 있어 실제 외부 데모 사이트와 연결 안됨
- **현재값**: `/demo/hankook/`
- **필요값**: `https://uttec-sensor.duckdns.org/admin` 또는 개별 데모 URL
- **해결방안**: 각 업체별 실제 데모 URL 확인 후 업데이트 필요

### 2.2 사이트 구조 관련 문제

#### 2.2.1 /platforms/career 페이지 누락
- **문제**: business-areas.ts에서 `link: '/platforms/career'`로 설정되어 있으나 해당 페이지가 존재하지 않음
- **현재 존재하는 페이지**:
  - /platforms/education
  - /platforms/certification
- **해결방안**: /platforms/career 페이지 생성 또는 링크를 /platforms 로 변경

#### 2.2.2 eduProject 사이트 연동
- **문제**: 사용자가 언급한 eduProject 사이트(http://178.128.90.37:5000/, http://178.128.90.37:8000/)가 홈페이지에 반영되지 않음
- **해결방안**: 진로지도 영역에 추가 링크로 포함하거나, 별도 섹션 구성

### 2.3 디자인/UX 관련

#### 2.3.1 참고사이트 트렌드 미반영
- **문제**: 참고사이트.md에 명시된 2025-2026 디자인 트렌드 일부 미적용
- **주요 미적용 항목**:
  - Scroll-driven Animations
  - Spatial Design (3D 요소)
  - Micro-interactions
- **해결방안**: 단계적 UI 개선 필요

### 2.4 통합 관련

#### 2.4.1 서비스 간 통합 내비게이션
- **문제**: 각 외부 서비스(자격시험, 교육 등)에서 UTTEC 메인 홈페이지로 돌아오는 링크 부재
- **해결방안**: 각 서비스에 공통 헤더/푸터 적용 또는 Back to Main 버튼 추가
