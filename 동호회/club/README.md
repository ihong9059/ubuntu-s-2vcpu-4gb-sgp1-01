# 배드민턴 동호회 웹사이트

## 설명
배드민턴 동호회 종합 관리 웹사이트

## 포트
8000

## 기본 관리자 계정
- 이메일: admin@club.com
- 비밀번호: admin123

## 주요 기능

### 회원 관리
- 회원가입 (관리자 승인제)
- 프로필 관리 (사진, 자기소개)
- 등급 관리 (A~E급)
- 권한 관리 (관리자, 운영진, 일반회원)

### 게시판
- 공지사항 (운영진만 작성)
- 자유게시판
- 가입인사
- 건의사항
- 댓글 기능

### 갤러리
- 앨범 생성
- 사진 업로드/삭제

### 일정 관리
- 일정 등록
- 참석 응답 (참석/불참/미정)

### 대진표
- 등급별 인원 입력 방식
- 실력 균형 매칭
- 경기 유형별 게임수 설정
- 기호-이름 매칭 기능

## 파일 구조
```
club/
├── app/
│   ├── __init__.py     # Flask 앱 초기화
│   ├── models/         # 데이터베이스 모델
│   └── routes/         # 라우트 (URL 핸들러)
├── templates/          # HTML 템플릿
├── static/             # CSS, JS, 업로드 파일
├── config.py           # 설정
└── run.py              # 실행 파일
```

## 실행
```bash
python run.py
```

## 기술 스택
- Flask + Flask-Login + Flask-SQLAlchemy
- SQLite 데이터베이스
- Bootstrap 5
